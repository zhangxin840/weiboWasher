微博洗刷刷
===========

去广告

隐藏不想要的栏目

清理提示

不改变微博原貌

清理规则实时更新

方便，开源，安全


安装方法
------------

* 360浏览器，360急速浏览器，猎豹浏览器：

  [插件在线安装](https://www.google.com)

  (或右键另存，拖入浏览器窗口安装)


* Chrome浏览器：

  1. [右键另存此链接](https://www.google.com)

  2. 将另存的文件拖入 设置-扩展程序页面


开发者
----------

data / settings.json 为实时更新的配置文件

开发者可修改settings.json来更正过滤规则
